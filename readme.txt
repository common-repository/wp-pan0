== Description ==

pan0.net integration. enter [pan0 http://path.to.yourimg/name_of_image] into post/page to add the image as panorama.

You can use also attributes: enter [pan0 640x480 http://path.to.yourimg/name_of_image] to set width x height. 

No other options supported, the panorama has in default 100% width/height, empty title.

!!! DO NOT USE SPACE IN IMAGE PATH !!!

== Installation ==
1. Unzip plugin into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. enter [pan0 http://path.to.yourimg/name_of_image] into post/page to add the image as panorama.
== Frequently Asked Questions ==
=== How to change size of the panorama ===
enter [pan0 widthxheight path_to_image] to set the dimensions.

No way for now.
